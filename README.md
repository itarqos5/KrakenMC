# Kraken Minecraft Server

High-performance Minecraft 1.21.1 server engine utilizing SIMD optimizations and an ECS architecture.

## Features
- **Core:** Rust Nightly (SIMD optimized).
- **Network Engine:** Azalea (1.21.1 Support).
- **Database:** Sled (Binary/Postcard/Gzip).
- **Features:** Parallel Chunk Gen, ViaKraken Protocol Bridge, Hard Persistence.
